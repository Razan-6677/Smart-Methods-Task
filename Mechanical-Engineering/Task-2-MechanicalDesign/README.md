# Task 2 - Mechanical Design (3D Keychain)

## Task Description

The objective of this task is to practice 2D sketching and 3D extrusion by designing a custom keychain. The requirements include:

1. Choosing an image (Spiderman Logo) and sketching it in CAD software.
2. Adding a 4mm circular cut to serve as a keychain ring hole.
3. Extruding the 2D sketch by 2mm.
4. Exporting the final 3D model as an STL file.

## Design

The keychain was sketched and extruded into a 3D solid body using Onshape.
